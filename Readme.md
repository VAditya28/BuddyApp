Buddy Book is a full-stack e-commerce web application designed for book lovers. It allows users to browse, search, and purchase books online with a seamless shopping experience. The platform includes admin and customer dashboards, secure checkout, and real-time inventory management — making it a complete digital bookstore solution.

🚀 Features

🛒 User-Friendly Shopping Experience: Browse and purchase books with dynamic filtering and sorting options.

🔐 Secure Authentication: Register, login, and manage user accounts with role-based access (Admin & Customer).

📦 Admin Dashboard: Manage products, categories, orders, and users efficiently.

💳 Cart & Checkout: Add to cart, update quantities, and complete orders through a secure payment process.

📚 Book Management: CRUD operations for books with ISBN, author, price, and category details.

📈 Analytics (Optional): View sales and inventory insights through admin metrics.

☁️ Cloud Ready: Easily deployable on Azure or AWS with scalable database support.

🧩 Tech Stack

Frontend: ASP.NET Core MVC, Razor Pages, HTML5, CSS3, Bootstrap, JavaScript
Backend: ASP.NET Core Web API, Entity Framework Core
Database: SQL Server / Azure SQL
Cloud & Tools: Azure App Service, Visual Studio, GitHub Actions (CI/CD), Docker
Version Control: Git & GitHub
